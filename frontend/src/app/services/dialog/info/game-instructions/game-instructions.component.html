Game Instructions

Rules

When the ball reaches one of the game window's side limits, the player defending the opposite side wins a point.

The first player that wins 5 points wins the match.

Versions

There are two game versions.

  • Classic: Each player controls a paddle that can move up or down.

  • Hero: Each player controls a paddle that can move up or down, and a hero that can appear from the top or the bottom of the player's side. Each hero has unique trajectories and can be invoked once per point.
